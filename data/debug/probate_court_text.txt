 Important Notices about eAccess 
 The case information contained within this web site is generated from computerized records maintained by Summit County Probate Court and is deemed to be public information. While every effort is made to assure the data is accurate and current, it must be accepted and used by the recipient with the understanding that no warranties, expressed or implied, concerning the accuracy, reliability or suitability of this data have been made. The Court, Clerks of Court, their agents, and the developers of this web site assume no liability whatsoever associated with the use or misuse of the data contained herein. 
 NOTE:   CASE NUMBER Searches Only 
 The Case Number requires an exact match INCLUDING spaces.  The five (5) digit case number must include the leading zeros.  Example: 1900 ML 00015 Failure to follow the specified format will result in NO DATA returned. Any case issued prior to December 29, 1999 may be searched by  NAME  only. 
 NOTE:     REGARDING ONLINE  IMAGES 
 Images are available for all public Case Types.  Confidential docket entries will not show images; Clicking on the image icon will return an Error:  No Document Image Available. Marriage License images  CANNOT  be used as   certified copies. SOME IMAGES MAY NOT BE AVAILABLE ON LINE, HOWEVER, THE ENTIRE FILE IS AVAILABLE AT PROBATE COURT
